# 名簿管理システム

## 説明
基本的な機能を実装した名簿管理システムとなっています。

## 使用言語・環境等
- HTML
- CSS
- Bootstrap 3.3.7
- Ruby 2.6.3
- Ruby on Rails 5.1.7
- AWS Cloud9
- Heroku
- RSpec
- macOS

## 機能
- ユーザー登録・削除、ログイン・ログアウト(devise)
- 名簿の作成・詳細・編集・削除
- 名簿とカテゴリの紐付け
- 名簿の検索(ransack)
- 名簿のソート(ransack)
- ページネーション(kaminari)
